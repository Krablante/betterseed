# Features

BetterSeed keeps the public release surface tight and practical.

## Project Workflow

- Separate BetterSeed package
- Separate `Open project` entry in Home and editor toolbar
- Editable `.snpsd` save/open
- Save project stays in the editor
- Reopened projects stay editable
- Embedded source kept and `edits_cache` removed

## Export And Stability

- Stock photo picker restored for normal open
- Regular photo export still works after reopening a project
- No-resize full-res export fix
- `Export as -> SAVE` fix
- View edits delete/edit roundtrip fixes

## Editing Power-Ups

- Shared numeric controls with exact value entry and hold-repeat
- `.bslook` import/export
- Preview fidelity improvements
- Curves 99 support
- Configurable global range expansion and boundary setting

## Localization And Quality-Of-Life

- BetterSeed-only custom UI language defaults to English outside `ru`, with `ru` overrides
- Author channel link
